# Jpage
一个前端分页插件 （基于jquery）
##   使用
  >
  > * ### 1.引用jquery    
  >   <script type="text/javascript" src="jquery-3.1.0.min.js"></script>  
  >
  > * ### 2.调用分页插件  
  >       parentDiv       父块索引  
  >       childDiv        子块索引  
  >       onePageNumber   一页显示子块数量  
  >       var jpage=new Jpage(parentDiv,childDiv,onePageNumber)  
  >       jpage.createPage();
